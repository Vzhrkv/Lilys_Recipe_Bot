# Телеграм бот для поиска рецептов по ингредиентам.

## Краткое описание
1. Принимает список ингредиентов(минимум 3) и выводит ссылку на рецепт пользователю.
2. Можно запросить другой рецепт по тому же списку ингредиентов.
3. Рецепты не повторяются.
4. Пользователь может добавлять рецепты в избранное
5. Бот позволяет просмотреть избранные рецепты пользователя по запросу.
